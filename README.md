Fossa の Infra チームは、App チームと比べると リモートワークを選ぶメンバーが多い という特徴があります。
そのため、働き方が ややオンライン寄りになる ことで、どうしても コミュニケーションの頻度が下がりやすい という課題が出てきます。
頻度が落ちると、日々の業務で生じる 小さな認識のズレが積み重なりやすい ので、
そこでfossaは、Slack のカスタムレスポンスを使って Teams の作業部屋をすぐ立ち上げられる仕組み を取り入れています。
これにより、メンバー同士が 気軽に声を掛けあえるようになり、ちょっとしたすり合わせが自然と増える ようになりました。

また、もうひとつ自分大事にしているのが Slack のスタンプを積極的に使うこと です。
一見すると小さなアクションですが、リアクションが増えることで チーム内の雰囲気がよくなり、コミュニケーションもよりスムーズになる と感じています。
